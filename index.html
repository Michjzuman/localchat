Local by design

Runs on one Linux host over a Unix socket. No cloud dependency, no external relay.

Real user identity

The daemon maps each connection to the real Linux UID using SO_PEERCRED.

Hardened service

Systemd sandboxing defaults keep the daemon isolated and focused on one job.
